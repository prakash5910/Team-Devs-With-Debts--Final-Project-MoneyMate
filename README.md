# MoneyMate
MoneyMate is a personal budget tracking application, created for **CSC 317 Foundations of Software Development** by
**Devs With Debts**.
The app allows users to track income, expenses, montly statements, savings goals, and account settings through a multi-page kivy interface.

## How to Run The Application

Make sure the following are installed on your device:
– Kivy (2.3.1)
– Python (3.13.0)

# Clone the Repository
git clone https://github.com/prakash5910/Team-Devs-With-Debts--Final-Project-MoneyMate.git
cd Team-Devs-With-Debts--Final-Project-MoneyMate

# You can run it through the following methods.
1. Locate main.py, and press the run button.
Or
2. Navigate to the root folder, and in your terminal, type the following command:
python main.py
