# MoneyMate 💰

> *A dollar saved is a dollar earned.*

MoneyMate is a personal budget tracking application built for **CSC 317 – Foundations of Software Development** by **Devs With Debts**. It provides a clean, mobile-style interface for tracking income, expenses, savings goals, and monthly statements — all stored locally on your device.

---

## ✨ Features

- **Signup / Login** — Secure account creation and authentication
- **Dashboard** — Monthly income and expense overview with donut charts and goal progress bars
- **Statements** — Browse and filter your transaction history
- **Transactions** — Add, view, and categorize income and expenses
- **Goals** — Set savings goals and track your progress with expected completion dates
- **Settings** — Update account info, currency, theme, and notification preferences
- **Local Storage** — All data is saved to a local SQLite database — no internet required

## ⚙️ Requirements

Make sure the following are installed before running the app:

- Python `3.13.0`
- Kivy `2.3.1`

Install Kivy via pip if needed:

```bash
pip install kivy==2.3.1
```

---

## 🚀 How to Run

**1. Clone the repository**

```bash
git clone https://github.com/prakash5910/Team-Devs-With-Debts--Final-Project-MoneyMate.git
cd Team-Devs-With-Debts--Final-Project-MoneyMate
```

**2. Run the application**

```bash
python main.py
```

Or open `main.py` in your IDE and press the **Run** button.

> The app will open in a 360×720 window simulating a mobile phone screen.
